# Insight AI — Multimodal Emotion Recognition System

Insight AI is a cutting-edge platform designed to decode human emotion through multiple modalities. Powered by deep learning, it analyzes text, audio, image, and video signals in real time to provide quantified emotional insights.

## Features

- **Text Analysis**: NLP-driven linguistic pattern recognition to uncover underlying sentiment.
- **Audio Analysis**: Evaluation of emotional prosody through vocal tone and pitch extraction.
- **Image Analysis**: Deep learning facial expression detection based on Ekman's core emotions.
- **Video Analysis**: Real-time temporal micro-expression tracking via webcam.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or pnpm

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   cd insight-ai
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Technology Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Framer Motion
- **UI Components**: shadcn/ui, Lucide Icons
- **State Management**: TanStack Query (React Query)
- **Backend**: Python (Flask/FastAPI) - see `backend/` directory
